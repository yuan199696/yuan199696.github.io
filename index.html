聊聊前端

发表于 2020-08-07 分类于 后端开发 ， JAVA

开篇词

一个记录学习过程的个人小站！

这两天想着搭建一个个人博客网站，本来准备通过阿里云的镜像市场中使用word press来搭建，但看着自己快到期的服务器还是放弃了这个想法，因为经济实力不允许。于是乎，决定白piao一把。其实，GitHub、gitee等代码托管网站都有一个功能：提供一个免费的存放静态页面的模块，具体怎么做网上有大把教程，这里不再赘述！

第一个系列决定写一些关于SSM框架（Spring、SpringMVC、Mybatis）的例子。主要的想法是：作为一个后端开发人员在没有前端开发者的情况下有时需要自己写一些简单的前端界面，比如数据展示时需要数据表格，表单提交时需要表单样式等。所以，这个系列的文章重点将放在前端数据的渲染上。作者也是一个小白，且作者姓氏为白，所以是一个真小白。

聊聊前端

其实，我一直对前端有一定的热爱，觉得样式布局以及实现方式比较好玩。所以，在业余接触过前端的一些知识，比如html、css、js、jQuery、VUE、微信小程序等，但都仅限于会用一些基础的东西。其次 ，还有一些前端好用的框架，比如bootstrap、layui、elementUI等等。对于我来说，学习前端可以在一定程度上起到陶冶情操的作用​​。建议大家有时间也可以去学学前端。像如bootstrap、layui之类的框架对于后端开发者来说比较好上手，可以快速的开发出一个界面并配合后端接口进行数据显示，这个系列的文章在前端部分主要运用layui，因为我觉得它比bootstrap原生界面要更加丝滑一些。

文章安排

文章的前几部分会简单的展示后端接口和前端界面该如何配合来进行数据展示，之后的部分会详细说明SSM框架如何与layui配合的一些细节以及具体用法，暂时就先安排这些，以后想到了再慢慢写。
