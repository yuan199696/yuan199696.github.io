# Linux 下安装 MySQL5.7

Linux 系统：CentOS8

MySQL 版本：MySQL5.7

将下载的 MySQL 传输到 Linux 下并解压到指定目录，然后重名

tar -zxvf xx.tar.gz -C /usr/local
cd /usr/local
mv xx.mysql mysql5.7

创建用户和组并初始化 mysql

配置 mysql 的配置文件

[root@localhost fan]# vi /etc/my.cnf
[client]
default-character-set=utf8
port=3306
[mysqld]
basedir=/usr/local/mysql
datadir=/usr/local/mysql/data
port=3306
character-set-server=utf8
collation-server=utf8_general_ci
lower_case_table_names=1
#重新定义错误日志输出目录
log-error=/usr/local/mysql/data/error.log

添加软连接到系统（相当于快捷方式便于执行）

修改 mysql 密码

开放 mysql 远程连接

将 mysql 服务添加到开机自动启动列表中

将 mysql 添加到环境变量中
